Current keyboard buttons:

Press K to shoot.
Press 1 to reload/switch weapons.
Press L to aim.
Press J to throw grenade.
Press and hold W to hold click and then move the mouse in the direction you want to walk.
Release W button to stop walking

Press and hold T then move the mouse to look around the room in the game.
Release T key to stop looking.

Steps to install:

1. Download the script .txt file from above.
2. Download AutoHotKey version 2 from 
2.. website. https://www.autohotkey.com/v2/
3. Install AHK from your downloaded file.
4. Rename the extension file from .txt to .ahk
5. Double click the .ahk file you renamed and...
6. Enjoy!!
